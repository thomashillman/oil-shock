## Learnings

- Repository is currently minimal and only contains a basic README.
- Existing user draft establishes a practical deployment baseline: Cloudflare Workers + D1 backend and Vercel frontend with preview/production separation.
- Main uncertainty is signal quality calibration, not infrastructure feasibility.
- MVP requirements are now concrete enough to proceed to architecture design and task decomposition.
- Design confirms precomputed snapshot serving is the key simplification to meet latency and reliability targets.
- Task plan should prioritize proving ingest->score->API coherence before expanding frontend/CI complexity.
- Using `corepack pnpm` is required in this environment because `pnpm` is not globally available on PATH.
- Local D1 schema application works with `wrangler d1 execute ... --local` using current config.
